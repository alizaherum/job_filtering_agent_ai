Job Filtering Agent
A smart, multi-source job filtering system that helps job seekers find relevant opportunities by applying personalized filters across multiple job platforms (LinkedIn, Glassdoor, Indeed, etc.).

Overview
The Job Filtering Agent is a Python-based intelligent job matching system that filters job listings based on user preferences, scores matches for relevance, and provides detailed summaries of the best opportunities. It's designed to integrate with various job data sources while maintaining flexibility for custom additions.

Key Features
Multi-Criteria Filtering – Filter jobs by location, job type, salary range, required skills, preferred companies, and experience level

Smart Scoring System – Each job receives a relevance score based on how well it matches user preferences

Source Tracking – Track where each job came from (LinkedIn, Glassdoor, Indeed, etc.) and filter by preferred sources

Company Rating Support – Include company ratings (e.g., from Glassdoor) as a filter criterion

Keyword Matching – Include or exclude specific keywords from job titles and descriptions

Remote Job Support – Special handling for remote and hybrid positions

Detailed Job Summaries – Generate formatted summaries with match scores, reasons, and key skills
